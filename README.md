# github-batch-clone

> A command line tool to batch clone github reps include yours and starred reps.

## Installation

```console
$ go get github.com/jeremaihloo/github-batch-clone
```

## Usage

```console
$ github-batch-clone --help
```

## Warning

If you starred a lot of reps, its very large to save !
